## AWS SimuLearn – Database Modernization & High Availability

I completed an AWS SimuLearn scenario where I helped a fictional insurance company reduce operational overhead for its database administrators and improve overall database availability and efficiency. I analyzed the customer's requirements and designed a managed database solution on AWS that minimized manual tasks like patching and infrastructure maintenance.

After finalizing the design, I built the solution step‑by‑step in a live AWS Console environment. I compared different AWS database types and their use cases, implemented Multi‑AZ deployments for high availability, and configured Amazon RDS read replicas to optimize performance. I also evaluated migration strategies using AWS Database Migration Service (AWS DMS) and reinforced the differences between managed and self‑hosted database approaches.

This simulation strengthened my practical skills in designing resilient, scalable, and low‑maintenance database architectures on AWS while practicing real‑world communication and problem‑solving with AI‑driven customer interactions.
