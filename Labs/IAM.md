##  Install & Configure the AWS CLI

I completed an AWS SimuLearn lab where I installed and configured the AWS CLI on a Red Hat EC2 instance and connected it to an AWS account. I accessed the instance over SSH, installed the AWS CLI manually, and verified the installation through version and help commands.

I reviewed IAM settings in the AWS Console, retrieved the access key and secret key provided in the lab details, and used them to configure the CLI with `aws configure`. After setup, I used the AWS CLI to interact with IAM, including listing users and retrieving policy information. I also completed a challenge to download a customer‑managed IAM policy in JSON format using AWS CLI commands.

This lab strengthened my understanding of AWS CLI installation, IAM authentication, and programmatic access to AWS services.
